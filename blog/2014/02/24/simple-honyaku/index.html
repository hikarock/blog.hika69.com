Chrome 拡張「シンプル翻訳」を更新

Feb 24, 2014 #javascript #chrome-extension

Chrome 拡張機能の「シンプル翻訳」を2年ぶりに更新しました。

素敵なバナーやアイコンは @meganejunkie 先生に作ってもらいました (ありがとうございます) 。

利用しているAPIを変更しましたが機能的にはほぼ変わっていません。

  • Google翻訳APIから、Bing Translator APIに変更
  • オプションから個別にBing Application IDが設定可能に
  • 翻訳結果の読み上げ機能(英語のみ)を追加

デフォルトだと一つのBing Application IDをみんなで使いまわすので、API制限かかるかもしれません。 なので一応「オプション」画面から個別にBing Application IDを上書きできるようにしています。

インストールはこちらから。

Chrome ウェブストア - シンプル翻訳
