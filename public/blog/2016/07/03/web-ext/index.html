web ext

Jul 3, 2016 # #

web-ext をインストール。

% node -v
v6.2.2
% npm install -g web-ext

> node-pre-gyp install --fallback-to-build

gyp ERR! configure error
gyp ERR! stack Error: Python executable "/Users/hikarock/.pyenv/shims/python" is v3.5.1, which is not supported by gyp.
gyp ERR! stack You can pass the --python switch to point to Python >= v2.5.0 & < 3.0.0.

gyp のインストールが走って Python のバージョンがいかんと怒られた。pyenv で 2.7 に切り替える。

% pyenv version
3.5.1 (set by /Users/hikarock/.pyenv/version)
% pyenv global system
% python --version
Python 2.7.10

再度 npm install -g web-ext を実行したらうまく入った。

% web-ext --version
{ Error: Cannot find module './build/Release/DTraceProviderBindings' ...
{ Error: Cannot find module './build/default/DTraceProviderBindings' ...
{ Error: Cannot find module './build/Debug/DTraceProviderBindings' ...

しかし実行すると DTraceProviderBindings がみつからない云々言われる。 一応エラーの後でバージョンは表示されていたけど気持ちわるいので --no-optional つきで再インストールしてみる。

% npm install -g --no-optional web-ext

上記のエラーはでなくなった。

次は web-ext コマンドを使って Chrome 拡張を Firefox で起動してみる。

% cd projects/simple-honyaku
% web-ext run
...
run: WebExtError: Temporary add-on installation is not supported in this version of Firefox (you need Firefox 49 or higher). For older Firefox versions, use --pre-install

Firefox 49 以上を使えと言われたので以下からダウンロードする

Firefox Developer Edition — Mozilla
